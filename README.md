## 简介

- 主控采用全志 V3S，单核 Cortex-A7，64MB 内存
- 板子带有 rj45 网口、耳机、麦克风、tf卡槽、USB、40pin RGB 屏幕接口
- 4.3 寸 800 x 480 分辨率屏幕
- 外壳为亚克力（使用 Shell.dxf 文件可以在某宝定制切割 2mm 厚度亚克力），M2.5螺丝固定
